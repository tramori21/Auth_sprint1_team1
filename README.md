## Обработка ошибок
Типовые ответы:
- 400/422 — неверные входные данные.
- 401 — невалидный access/refresh токен (истёк, неверная подпись, неверный формат), либо неверные учётные данные.
- 403 — недостаточно прав (RBAC).
- 404 — сущность не найдена.
- 409 — конфликт (например, логин уже занят).
- 500 — внутренняя ошибка сервиса.
